# HayFrp CLI

HayFrp CLI 是一个命令行工具，用于管理 HayFrp 内网穿透服务。

## 功能特性

- **用户管理**：登录、注册、签到、查看信息、重置密码等
- **隧道管理**：创建、编辑、删除、列表、配置文件获取、状态切换等
- **节点查询**：节点列表、节点信息、服务统计等
- **自动登录**：保存登录状态，下次自动登录
- **自动下载**：自动下载对应平台的 frpc 并启动
- **API 容灾**：多端点自动故障转移
